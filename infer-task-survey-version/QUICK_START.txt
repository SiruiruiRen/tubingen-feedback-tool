🚀 FASTEST WAY TO USE (3 steps)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Double-click "index.html" to open in your browser

2. Fill out the form:
   • Your name
   • Select video
   • Paste reflection text
   
3. Click "Generate Feedback"
   • Choose Extended or Short style
   • Wait 30-60 seconds
   • Read your personalized feedback!

⚠️  IMPORTANT: The CORS proxy must be running
    (It's already deployed on Render and working!)

🆘 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: "Failed to fetch" or CORS error
→ Wait 30 seconds for Render proxy to wake up
→ Try again

Problem: No feedback appears
→ Check browser console (F12) for errors
→ Make sure you filled in name and video
→ Reflection should be at least 20 characters

Problem: Wrong language
→ Click language toggle in Settings panel
→ Regenerate feedback if needed

Problem: Page looks broken
→ Clear browser cache (Ctrl+Shift+R)
→ Make sure all files are in same folder
